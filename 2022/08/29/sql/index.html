sql查询语句

发表于2022-08-29|更新于2025-01-09|sql

多段代码需要在每句末加;单个不需要
运行原理：where-group by-having-order by-limit-select

select & from

where

order by （排序）

limit

聚合函数&group by

having

having是分组聚合后筛选（只能用分组后得到的字段）；where是分组聚合前筛选

运行原理（总结）

from语句从数据库中调取复制一份表格
where语句在复制的表格中筛选出符合条件的数据行
group by语句依据指定字符对筛选后的数据分区，将依据得到字段去重复，相当于Excel建立了一个数据透视表，添加了行标签
having语句筛选满足条件的分组
order by语句对筛选后的数据排序
limit语句对排序后的数据限制显示的行
select语句，提取最后要显示的字段

四舍五入函数round(x,y)

对x进行四舍五入，精确到小数点后y位 round(3.15,1)=3.2

窗口函数(having后才可以引用窗口函数的结果)

over(partition by 字段名 order by 字段名 asc|desc) partition by指定分区依据，order by指定排序依据
注意和group by的区别
1.窗口函数写在select后面；2.partition by只分区不去重，不分区则整个表一个区；3.order by是必选项，在分区内排序。

排序赋值方法（给排名）

rank():跳跃式排序，例如99，99，98，97给排名1，1，3，4
dense_rank():并列连续型排序，例如99，99，98，97给排名1，1，2，3
row_number():连续型排序，例如99，99，98，97给排名1，2，3，4（不允许并列）

rank()over(partition by 字段名 order by 字段名 asc|desc)

偏移分析函数

lag/lead(字段名,偏移量[,默认值])over(partition by 字段名 order by 字段名 asc|desc) #lag向上取，lead向下取
通常用于做减法得出增长值，如计算每周一新增xxx
select name, date_format(whn,’%Y-%m-%dd), (confirmed-lag(confirmed,1)over(order by whn)) from covid where name=’Italy’ and weekday(whn)=0 order by whn #weekday(whn)=0表示周一

表连接

内/左/右连接
select 字段名
from 表名1 inner/left/right join 表名2 on 表名1.字段名=表名2.字段名 #内连接inner可以忽略（默认）

select team1, team2, player from game join goal on game.id=goal.matchid where player like ‘Mario%’ #根据id连接表

表格也可用别名：from game ga join eteam et on et.id=ga.team1

多表连接时，字段名唯一可以不说明在哪个表

子查询

1.子查询本身就是一个完整的查询，用括号包裹在主查询中；2.子查询的结果返回给主查询；3.子查询可以在select, from, where, having子句中使用，但注意不同句子能接受的子查询种类有差别；4.子查询可以多重嵌套
